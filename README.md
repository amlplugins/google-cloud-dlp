# @amlplugins/google-cloud-dlp

Native `@google-cloud/dlp` SDK re-exported under the `@amlplugins` namespace.

> Cloud Data Loss Prevention.

## Install

```bash
npm install @amlplugins/google-cloud-dlp
```

## Use

```ts
import { /* named exports from @google-cloud/dlp */ } from "@amlplugins/google-cloud-dlp";
// or
import sdk from "@amlplugins/google-cloud-dlp";
```

Built and published by GitHub Actions on every push to `main`.

Part of the amlplugins family — see https://github.com/amlplugins.
